# Trading Dashboard

Streamlit Dashboard für alle 5 Trading-Strategien.

## Setup

1. Repo auf GitHub hochladen
2. streamlit.io/cloud → New app → Repo verbinden
3. Secrets eintragen: EODHD_API_KEY

## Dateien aktualisieren

Aus Colab automatisch via sync_to_github() Funktion.

## Strategien
- Kassandra (Länder-ETF)
- S&P 100 Momentum
- IVY / Hybrid-RAA
- ETF Aktien
- Small Cap Europe
